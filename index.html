Jrid

A dynamic grid overlay in JavaScript.

Install

 
yarn add jrid

Usage

Build System

 
import {Jrid} from 'jrid'
const jrid = new Jrid()

Script Tags

 
<script src="https://unpkg.com/jrid"></script>
<script>
const jrid = new Jrid(
  document.getElementById('jrid'),
  {
    locale: 'fr-CA',
    fpsLimit: 10
  }
)
</script>
